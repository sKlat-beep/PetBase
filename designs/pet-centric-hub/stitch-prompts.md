## Variant Generation Instructions

After generating Screen 1 (Pet Overview), generate 2 variants using:

```json
{
  "tool": "generate_variants",
  "arguments": {
    "projectId": "[NUMERIC_PROJECT_ID]",
    "screenIds": ["[SCREEN_1_ID]"],
    "variantOptions": {
      "creativeRange": "EXPLORE",
      "aspects": ["LAYOUT", "COLOR_SCHEME"]
    },
    "numVariants": 2
  }
}
```

This uses creativity level ~0.7 (EXPLORE) to produce alternative layout and color interpretations while keeping the core content and earth-tone identity intact.
